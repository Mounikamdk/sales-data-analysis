# Sales Data Analysis

## 📊 Project Overview
This project analyzes sales data using Python.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## 📈 Analysis Performed
- Total sales calculation
- Sales by product
- Sales by region
- Highest and lowest performing products
- Visualization of sales trends

## 💡 Business Insights
- Laptop has highest sales
- Tablet has lowest sales
- USA region performs best
